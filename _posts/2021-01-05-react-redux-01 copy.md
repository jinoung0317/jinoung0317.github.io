## Redux 사용법

<br/>

### Redux npm donwload
```shell
 npm install redux react-redux redux-action
```
  - redux : redux에 대한 base lib
  - react-redux : react에서 redux 사용지원 lib
  - redux-action : redux 사용편의지원 lib
